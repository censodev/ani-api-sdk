# AniAPI Java SDK
## Introduction
A Java SDK for [AniAPI](https://github.com/AniAPI-Team/AniAPI) REST APIs

## Prerequisites
* Java 11
* Maven 3

## Installation
### Config in pom.xml
```
<dependency>
    <groupId>io.github.censodev</groupId>
    <artifactId>aniapi-java-sdk</artifactId>
    <version>1.0.0</version>
</dependency>
```
### Install with Maven CLI
```
mvn install
```
